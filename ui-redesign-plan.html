一顆會變色的大按鈕就是整個 app：綠 = 開放（點了上鎖）、紅 = 鎖定（點了掃卡解鎖）。

✅ 已完工並驗收 — 2026-07-09 · 測試 14/14 綠 · Release build 過 · 已裝機

設計原則

  • 顏色即狀態：綠圓 = 未鎖、紅圓 = 已鎖。不用讀字就知道現在什麼狀態。
  • 圓即按鈕：點綠圓 → 立即上鎖（上鎖不需要卡，沒有理由多一步）；點紅圓 → 叫出 NFC 掃卡解鎖（解鎖必須有實體 key，這是產品本體）。
  • 次要資訊降級：配對 key 數量縮成小 chip、狀態訊息縮小、大按鈕以外不再有第二顆主要按鈕。
